Tus Finanzas App

Tus finanzas App es una aplicacion que permite poder registrar todos los ingresos y gasto que realizamos en la vida cotidiana. Tiene como objetivo permitirle a los usuario poder administrar sus finanzas personales de una manera practica y sencilla.

La aplicacion consta de una API desarrollada en Node. Dicha API utiliza Prisma ORM para realizar las consultas a la base de datos. Por otro lado, el frontend fue desarrollado con Angular y Css puro.

Projects App

Projects App es una aplicacion en desarrollo, que permite la creacion de diferentes proyectos y sus tareas, con el objetivo de poder llevar el seguimiento del mismo. Permite crear nuevos proyectos y tareas, asi como tambien marcar las tareas como completadas.

La aplicacion esta desarrollada en Node y su motor de plantillas PUG. Ademas se utilizo Sequelize ORM para realizar las consultas a la base de datos.
